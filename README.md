# b站屏蔽增强器

b站屏蔽指定用户或者关键词视频

根据用户名、uid和视频关键词进行屏蔽视频，作用场所，频道，首页推荐，搜索页，播放页右侧推送

需要添加是可以修改js文件中的userNameArr和userUIDArr以及videoNameArr，有注释描述
对着注释对应的数组后面添加即可
